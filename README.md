# Qyvera AI

> Your Personal AI Productivity Hub — Access 500+ AI models for free

Qyvera AI is a modern AI chat platform powered by [Puter API](https://puter.com). It provides access to 500+ AI models from major providers through a single, unified interface — no API keys required.

## ✨ Features

- 🤖 **20 Pre-configured Top Models** — GPT-4o, Claude Sonnet 4, Gemini 2.5, Llama 4, DeepSeek R1, Grok 3, and more
- 🆓 **Free to Use** — Powered by Puter's User-Pays model, no API keys needed
- 🌐 **500+ Models Available** — Access models from OpenAI, Anthropic, Google, Meta, Mistral, xAI, and dozens more
- 🎨 **Beautiful UI** — Modern, responsive interface with dark mode
- 💬 **Chat History** — Persistent conversations stored locally
- 🔌 **Plugin System** — Extend functionality with plugins
- 📱 **PWA Support** — Install as a desktop/mobile app

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or pnpm

### Development
```bash
npm install
npm run dev
```

### Build & Deploy to Firebase
```bash
npm run build
firebase deploy
```
